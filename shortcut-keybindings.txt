1. How to close a tab?
	command + w

2. How to close a application?
	command + q

3. How to refresh web page?
	command + r

4. How to lock the computer?
	Press power button for two seconds or close the led

5. PageUp/Down?
	option + ArrowUp or ArrowDown

6. PageHome/End?
	command + ArrowUp or ArrowDown

7. FN keys
	F3: expo
	F4: launchpad
	F5/F6: dim off or light on keyboard
